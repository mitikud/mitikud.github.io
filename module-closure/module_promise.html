  2. Define Global Scope and Local Scope in Javascript.
    • Global Scope
      • It is a scope outside a given function
      • A variable assigned befor it declared has global scope
      • In webpage the global variables are deleted when the browser is closed
    • Local Scope
      • It's a scope within a function
      • only accessible from within the function
      • local scop can access from their enclosing functions and other global scopes.
      • Variables declared inside a function have local scope and they are created when the function starts and deleted when the function completed
